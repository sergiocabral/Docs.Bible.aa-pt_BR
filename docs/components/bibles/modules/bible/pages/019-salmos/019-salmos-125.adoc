[[v19-125-0]]
=== Salmos, 125
// aa ?, AA - Almeida Revisada Imprensa Bíblica

[[v19-125-1]]

^<<v19-125-1,1>>^ Aqueles que confiam no Senhor são como o monte Sião, que não pode ser abalado, mas permanece para sempre.

[[v19-125-2]]

^<<v19-125-2,2>>^ Como estão os montes ao redor de Jerusalém, assim o Senhor está ao redor do seu povo, desde agora e para sempre.

[[v19-125-3]]

^<<v19-125-3,3>>^ Porque o cetro da impiedade não repousará sobre a sorte dos justos, para que os justos não estendam as suas mãos para cometer a iniqüidade.

[[v19-125-4]]

^<<v19-125-4,4>>^ Faze o bem, ó Senhor, aos bons e aos que são retos de coração.

[[v19-125-5]]

^<<v19-125-5,5>>^ Mas aos que se desviam para os seus caminhos tortuosos, levá-los-á o Senhor juntamente com os que praticam a maldade. Que haja paz sobre Israel.
