[[v19-131-0]]
=== Salmos, 131
// aa ?, AA - Almeida Revisada Imprensa Bíblica

[[v19-131-1]]

^<<v19-131-1,1>>^ Senhor, o meu coração não é soberbo, nem os meus olhos são altivos; não me ocupo de assuntos grandes e maravilhosos demais para mim.

[[v19-131-2]]

^<<v19-131-2,2>>^ Pelo contrário, tenho feito acalmar e sossegar a minha alma; qual criança desmamada sobre o seio de sua mãe, qual criança desmamada está a minha alma para comigo.

[[v19-131-3]]

^<<v19-131-3,3>>^ Espera, ó Israel, no Senhor, desde agora e para sempre.
